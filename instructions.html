Checkout Instructions

Prerequisite: install Mercurial.

Command-Line

hg clone https://javadoc.google-api-java-client.googlecode.com/hg/ javadoc

Eclipse 3.5

Prerequisite: install Eclipse, and the Mercurial plugin.

  • Checkout Mercurial project
    • File > Import...
    • "Select"
      • Select "Mercurial > Clone Existing Mercurial Repository"
      • Click on "Next"
    • "Clone repository"
      • URL: https://javadoc.google-api-java-client.googlecode.com/hg/
      • Clone directory name: javadoc
      • Click on "Next"
    • "Select working directory revision"
      • Click on "Next"
    • "Import Projects"
      • Click on "Finish"
